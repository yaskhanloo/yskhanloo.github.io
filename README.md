# Portfolio

Personal portfolio website showcasing AI/ML projects with a focus on healthcare NLP and medical imaging.

## Tech Stack

- React 19
- Vite
- React Router
- Modern CSS

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
npm run deploy
```
